# Lazarus_IDE_AddIns

## Lazarus Object Inspector Pal (LazObjInspPal)

How to install and use

- Open the lpk file in Lazarus "menu - package - open package file (.lpk)"
- Compile the package (Compile Button in Package Window)
- Install the package (Use Dropdown and Install command)
- Wait for Lazarus being recompiled and reopened
- Use new Button in Tools - menu to start config dialog
- You might wish to configure a coolbar to contain the button
