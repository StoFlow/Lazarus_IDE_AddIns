# Lazarus_IDE_AddIns

## Lazarus for Windows Colored Editor Tabs (LazWinColEdTabs)

### Config examples

To use one of these config files
(while the add-inn is installed),
copy it into the folder
C:\Users\[username]]\AppData\Local\lazarus
and rename it to
LazWinColEdTabs.xml
(remove the "_..."- suffix)
and then restart Lazarus.
