# Lazarus_IDE_AddIns

## Lazarus for Windows Colored Editor Tabs (LazWinColEdTabs)

### Config examples

To use one of these config files
(while the add-inn is installed),
copy it into the appropriate Lazarus Configs folder
e.g. :
C:\lazarus4.0.cfg
or
C:\fpcupdeluxe\config_lazarus

and rename it to
LazWinColEdTabs.xml
(remove the "_..."- suffix)
and then restart Lazarus.
